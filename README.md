# Food App

This is a food ordering app.
## Features

* [x] Android and iOS

  * [x] Fetch Data from Firestore
  * [x] Beautiful UI
  * [x] Item Add To Cart
  * [x] Item Add To Favorite
  * [x] Order the food which is added in cart
